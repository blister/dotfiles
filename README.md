# Installation
```console
sudo apt update
sudo apt upgrade
https://ppa.launchpadcontent.net/git-core/ppa/ubuntu
sudo add-apt-repository ppa:neovim-ppa/stable
sudo add-apt-repository ppa:git-core/ppa
sudo apt install neovim
sudo apt install git
sudo apt install tmux
```
